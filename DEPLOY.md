## Один раз включить GitHub Pages (2 минуты)

1. Откройте: https://github.com/yersat47/ai-business-os/settings/pages
2. В **Build and deployment** → **Source** выберите: **Deploy from a branch**
3. **Branch**: `gh-pages` → папка `/ (root)` → **Save**
4. Подождите 1–2 минуты — сайт появится по ссылке выше

После этого каждый push в `main` автоматически обновляет сайт.

## Альтернатива: Vercel (ещё проще)

1. Откройте: https://vercel.com/new/clone?repository-url=https://github.com/yersat47/ai-business-os
2. Нажмите **Deploy** (войдите через GitHub)
3. Получите ссылку вида `https://ai-business-os.vercel.app`
