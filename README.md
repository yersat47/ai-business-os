# AI Business OS

AI-powered Decision Intelligence Platform for SMB companies.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000/ru](http://localhost:3000/ru).

**Stable local setup (no Cloudflare tunnels):** see [LOCAL_SETUP.md](./LOCAL_SETUP.md) (RU).

**Permanent demo URL (GitHub Pages):** https://yersat47.github.io/ai-business-os/ru/

**Windows:** double-click `scripts/start-local.bat`

## Demo flow

1. Landing page → **Get Started Free**
2. Register with any email/password
3. **Create company** → complete the 11-step wizard
4. Dashboard loads with Urban Mode mock data

**Join company:** use invite code `URBAN1`

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run start` — production server
- `npm run lint` — ESLint
